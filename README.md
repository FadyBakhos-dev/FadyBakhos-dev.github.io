# Fady Bakhos — Portfolio Website

A personal portfolio site for Fady Bakhos, an aspiring web developer transitioning from a career in fitness coaching into building practical web applications.

## Live Site

Deployed via GitHub Pages: `https://FadyBakhos-dev.github.io/`

## Tech Stack

- **HTML** — page structure
- **CSS** — styling, layout, and responsive design (no frameworks)
- **JavaScript** — scroll-based nav highlighting and small UI interactions

## Purpose

This portfolio exists to:

- Introduce Fady as a developer and what he's building toward
- Show current skills (HTML, CSS, JavaScript, Git, GitHub) and what's next (React, Node.js, SQL)
- Showcase real projects, starting with a Fitness Progress Dashboard
- Provide a simple way for recruiters or collaborators to get in touch

## Project Structure

```
FadyBakhos-dev.github.io/

├── script.js          # Scroll highlighting + interactions
├── assets/

```

## Running Locally

No build step required. Clone the repo and open `index.html` in a browser, or serve it locally:

```bash
git clone https://github.com/FadyBakhos-dev/FadyBakhos-dev.github.io.git
cd FadyBakhos-dev.github.io
# then just open index.html, or use a simple local server
```

## Deploying to GitHub Pages

1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Under **Source**, select the `main` branch and `/ (root)` folder.
4. Save — your site will be live at `https://FadyBakhos-dev.github.io/`.
