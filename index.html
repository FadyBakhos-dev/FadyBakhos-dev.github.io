01 — About

Coaching people in the gym taught me how to break hard problems into small, repeatable sets. Code works the same way.

I'm Fady, a gym coach by trade, and a web developer in progress. I spend my days helping people track progress and hit targets, and I'm now applying that same mindset to building software.

I'm learning HTML, CSS, and JavaScript with a focus on real, usable tools rather than tutorials I'll never touch again. Everything I build starts from a problem I actually see on the gym floor, progress tracking, goal logging, simple dashboards and grows from there.

Next up: getting comfortable with React and basic backend work so I can take these projects from static pages to full applications.
